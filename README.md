# diablofresher

## Project setup

```
npm install
```

### Development (Vite)

```
npm run dev
```

### Production build

```
npm run build
```

Preview the production build locally:

```
npm run preview
```

### Lint

```
npm run lint
```

### GitHub Pages

The app is built with `base` set to `/diablo2fresher/` for production. Deploy the `dist` folder to GitHub Pages (for example `git subtree push --prefix dist origin gh-pages`).
